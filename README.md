# Webpack Starter

An ultralight webpack starter

### Setup

```
npm i
npm start
```

### Testing

```
npm test
```

### Build Prod 

```
npm run build
```

### Features:

* ES6 Support via [babel](https://babeljs.io/) 
* Linting via [eslint-loader](https://github.com/MoOx/eslint-loader)

Inspired by [wbkd/webpack-starter](https://github.com/wbkd/webpack-starter)
